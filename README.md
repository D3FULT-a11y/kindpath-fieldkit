# KindPath Field Kit

Public working materials for KindPath Collective:
- Whitepaper drafts
- Field Kit appendices
- Diagrams and supporting assets

## Intent
This repo exists to make the work reviewable, citeable, and improvable — without turning collaboration into chaos.

## How to contribute
- Use **Issues** to raise questions, critiques, and proposals
- Use **Pull Requests** to suggest changes (small and traceable)
- If you’re unsure, open an Issue first

## Structure (recommended)
- /whitepaper
- /field-kit
- /diagrams
- /references
- /licenses
